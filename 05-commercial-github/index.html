5. 付费服务¶

GitHub神奇的协同工具使得开源项目的创建和协同更加简单、高效。有些人可能会提出疑问：能否把GitHub用于私有项目呢？即能否只允许指定的用户访问项目和版本库，而其他人不能访问呢？能否在企业内部架设一个一模一样的GitHub服务呢？GitHub针对这些需求提供了解决方案，这些解决方案需要或多或少地付出一定费用。
